# Bluetooth Deauthenticator [blue-deauth] v20200425-1
A simple script that makes possible BLE deauthentication!

### Simple usage

`root@kali:/scripts/blue-deauth# bash blue_dos.sh FC:58:FA:XX:XX:XX 600`

### Requirements:
```
l2ping (comes with Kali)
```
